## 一句话理解

字体类型 + 笔画结构 + 材质光效 = 高级字体设计。

---

## 核心特点

- 多种字体风格（时尚/书法/复古/未来等）
- 详细的笔画/结构/重心描述
- 材质效果（金属/玻璃/旧印刷）
- 光影效果（霓虹/投影/高光）

---

## 核心模板

```
"文字内容"，字体类型，笔画/结构/重心/边缘描述；加入字效/材质/光影/背景，整体气质。
```

---

## 常用字体风格

### 时尚体
```
"高级玩家"，时尚体，字形修长，笔画干净，字距舒展，结构优雅，整体像时尚杂志标题。
```

### 毛笔书法
```
"人民日报"，报头题字毛笔体，字形庄重开阔，笔画厚实有墨色重量，带传统毛笔题词的顿挫和筋骨，整体像中文主流报纸报头。
```

---

## 变量说明

| 变量 | 说明 |
|------|------|
| 文字内容 | 要呈现的文字 |
| 字体类型 | 时尚体/毛笔字/衬线体等 |
| 笔画描述 | 粗细/结构/边缘等 |
| 材质/光效 | 金属/霓虹/投影等 |

---

## 变量使用指南

> ⚠️ 此风格不适用于IP KV套用

此风格为纯字体设计专用，主要关注字体本身的风格设计，不适合作为电商IP KV的整体画面风格。

---

## 风格锚点（必须保持）

- 字体风格明确
- 笔画描述具体
- 材质质感到位

---

## 🚫 避免风格

- 模糊描述（好看/高级）
- 空洞指令
