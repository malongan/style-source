# 撕纸美学

**标签**：#插画 #撕纸 #平面 #手绘 #文艺 #封面  
**触发词**：撕纸美学、撕纸风、纸面裂隙、手绘插画封面  
**适用场景**：书籍封面、文艺海报、插画封面、品牌视觉  
**比例**：16:9 横版  
**来源**：@xiaoxiaodong01  
**链接**：https://x.com/xiaoxiaodong01/status/2060007845903655125

## 一句话理解

高纯度结构色纸面撕出一道裂隙，内部世界从中显露，形成纯色空场压住内部世界的封面感。

---

## 核心特点

- 封闭纸面 + 不规则裂隙
- 高纯度结构色背景
- 手绘平涂 + 线描质感
- 单一强焦点 + 前景遮挡
- 克制出版物字体气质

## 完整模板

```
以{{具体主题内容}}为核心生成一张竖版平面插画海报，先用一个由{{主题情绪}}推导出的{{高纯度结构色}}铺满画面，让它像低噪声的彩色纸面一样承担遮挡、空白和呼吸；画面不建立真实景深和完整场景，而把背景处理成一层封闭表皮，表面只有轻微纸纹、扫描颗粒和细小印刷不均。主视觉不要完整陈列在背景前，而是在画面中上部由纸面撕出一道横向延展、略带斜势的不规则裂隙，裂口带白色纸纤维、暗色内缘、尖角毛边和轻微外翻感，像内部世界把纸面撑开。主题只通过被裁切的关键局部、可识别边缘、一个小而强的焦点和高密度碎片从裂隙内部显现；完整轮廓被纸面吞掉，次要信息被遮住，只留下最能说明主题的形体压力、材质触感和象征核心。焦点面积不大，却必须像凝视点一样支配第一眼阅读，被近景遮挡形体半包围；这个遮挡形体由主题语义转化而来，具有手势般的曲线、近距离触觉和横向引导力，覆盖焦点的一部分，让观看带有被拨开、被发现、被保护的阻力。裂隙边缘聚集由主题派生出的细碎形态，它们沿上下边和左右端形成软边界，局部外溢到纸面上，但不平均散开；碎片既是内容，也是裂口结构的一部分，应通过密度、方向和色点把内部世界推向外部空场。色彩保持清楚面积层级：最大面积{{结构色}}保持纯净，裂隙内部使用中等面积的{{主体生命色}}或材质色，小面积{{情绪亮点}}只在边缘碎片和焦点附近闪现，{{深色锚点}}压在焦点轮廓、内缘和关键遮挡处，最高明度只用于撕纸边、微小高光和材料切口。整体采用手绘平涂色块、清晰蓝黑或深色线描、轻微纸纹、扫描噪点和不均匀印刷颗粒完成，让对象像被纸面遮住后仍能凭局部轮廓、边缘取舍和局部密度被认出。下方或最安静的空白区保留大量纯色呼吸，文字若出现，只作为细致的题名、编号、短句和微型注脚，使用克制的出版物字体气质、细笔画、宽字距和低干扰层级，文字与空白协作，不抢裂隙内部的焦点。整张画面形成纯色空场压住内部世界、裂隙暴露主题秘密的封面感，稳定保留大片安静纸面、中上部破口、高密度内部、单一强焦点、前景遮挡、边缘碎片、暗色锚点和低声文字之间的秩序。
```

## 风格锚点（必须保持）

- 封闭纸面 + 不规则横向裂隙
- 高纯度结构色大面积的呼吸感
- 单一强焦点 + 前景半包围遮挡
- 手绘平涂色块 + 线描 + 纸纹质感

- 大面积文字干扰焦点
- 光滑无纹理的表面
