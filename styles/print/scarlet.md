## 一句话理解

白色纸张底 + 超大红色衬线字母 + 马克笔描边 = 故意的不完美。

---

## 核心特点

- 白色纸张底场
- 超大红色衬线字母
- 写实抠图主体
- 粗糙黑色马克笔轮廓描边
- 强对比非对称负空间

---


## 变量说明

| 变量 | 说明 |
|------|------|
| SUBJECT | 主要写实抠图对象 |
| SUBJECT_ACTION | 与红色字形的关系 |
| PRODUCT_OR_PROP | 次要道具 |
| LOCATION | 简洁编辑封面场 |
| MAIN_TEXT | 短虚构封面标题 |
| SECONDARY_TEXT | 作者、期号等 |
