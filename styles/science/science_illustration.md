图像类型：{{Graphical Abstract / 作用机制图 / 工作原理图 / 论文主图 / 实验流程图 / 技术路线图 / 解剖结构图 / 手术步骤图 / 病理机制图}}

输出结果：{{最终结果，可留空}}
关键视觉元素：{{3–10个必须出现的元素}}
结构形式：{{自动判断 / 左中右流程 / 上中下流程 / A→B→C→D步骤 / 主体+局部放大窗 / 微观-中观-宏观 / 解剖剖面 / 多面板图组}}
语言：{{中文 / 英文 / 中英混排 / 不要文字}}
比例：{{1:1 / 4:5 / 3:4 / 16:9 / 5:2 / 3:2}}
文字需求：{{少量标签 / 完整标注 / 不要文字}}
配色方案：{{Nature系克制机制配色 / Science系理性机制配色 / Cell系生命科学高对比配色 / 医学图谱浅色配色 / 外科解剖配色 / 自适应}}
禁用元素：{{不要出现的元素}}

请生成一张真正适合论文使用的科研插图。

它必须清楚回答：
1. 研究对象或解剖结构是什么？
2. 输入端或起始条件是什么？
3. 中间发生了什么机制、过程、变化或操作步骤？
4. 最终输出什么结果？
5. 关键机制发生在哪个位置、哪个结构、哪个细胞、哪个材料或哪个系统中？

【结构逻辑】
请根据主题自动选择最合适的科学叙事结构：
1. 输入 → 过程 → 输出 - 适合 Graphical Abstract、工作原理图
2. A → B → C → D 步骤流程 - 适合作用机制图、实验流程图、手术步骤图
3. 主体 + 局部放大窗 - 适合论文主图、结构解释图、细胞机制图、解剖细节图
4. 微观 → 中观 → 宏观 - 适合材料、生物医学、组织工程、纳米医学
5. 左侧问题 / 中间方法 / 右侧结果 - 适合论文图形摘要、技术路线图
6. 解剖剖面 / 结构分层 - 适合器官结构图、病理机制图、手术说明图

画面必须有明确方向。箭头、流线、编号、标签必须服务逻辑，不可乱用。

【论文插图视觉要求】
1. 图形清晰 - 每个关键结构都要可辨认，不要抽象到看不懂
2. 标签准确 - 标签要短、清楚、功能化，避免大段文字
3. 机制明确 - 如果涉及递送、结合、催化、释放、激活、抑制、修复、检测、传输、切除、缝合、吻合、植入，必须用图形路径表现
4. 局部放大 - 如有微观机制、解剖细节或关键操作点，请使用局部放大窗展示关键细节
5. 科学可信 - 不能为了好看牺牲基本科学逻辑。解剖结构、器官位置、组织层级、手术步骤和机制路径必须尽量合理

【配色方案 - 完整配色见 styles/_palette.md】
可选配色：Nature系克制机制配色、Science系理性机制配色、Cell系生命科学高对比配色、医学图谱浅色配色、外科解剖配色、自适应

【文字与标签规则】
1. 标题可有可无，视图像类型决定
2. Graphical Abstract 可使用短标题和阶段标签
3. 机制图必须有关键步骤标签
4. 解剖图必须保证结构标签清楚但不拥挤
5. 手术步骤图可使用编号、箭头和短句说明
6. 不要写大段解释性文字
7. 如果用户选择"不要文字"，则尽量通过视觉结构表达机制

【严格禁止】
- 只做漂亮科技图，没有机制
- 箭头乱飞 / 结构不清 / 元素堆砌
- 过度发光 / 游戏CG感 / 商业广告感
- 文字太多 / 标签不可读 / 配色普通蓝紫化
- 科学结构明显错误 / 解剖位置明显错误
- 手术步骤不合逻辑 / 真实期刊logo、真实品牌、真实机构标识
