## 一句话理解

高端时尚编辑视觉系统：4种版式方向 × 品牌调性变量 = 杂志级品牌KV/海报/封面。

---

## 核心特点

- 4种版式方向可选
- 品牌调性变量驱动设计
- 杂志级摄影 + 精致排版
- 强负空间、优雅构图
- 高端品牌感，非电商感

## 完整模板

```
Use this as a single-image prompt template.
If you want multiple images, generate each one separately. Do not merge them into one collage unless the chosen direction is specifically a lookbook collage spread.

Direction Type: [Giant typography campaign / Couture magazine cover / Accessories editorial page / Lookbook collage spread]
Brand Name: [BRAND]
Theme / Collection: [THEME]
Product / Focus: [PRODUCT OR SUBJECT]
Model: [MODEL DESCRIPTION or "no full model, product-focused"]
Brand Mood: [MINIMAL / QUIET LUXURY / MODERN / ROMANTIC / SHARP / COOL / SOFT / AVANT-GARDE]
Color Palette: [COLORS]
Main Headline: [HEADLINE]
Subheadline: [SUBHEADLINE]
Extra Small Text: [OPTIONAL SHORT TEXT]
Aspect Ratio: [9:16 / 4:5 / 3:4]

Create a high-end fashion editorial visual for [BRAND], based on the theme [THEME], with the focus on [PRODUCT OR SUBJECT]. The output should feel like a premium brand campaign / fashion magazine editorial / lookbook direction, with a strong sense of art direction, typography, layout, and brand mood. This is not a generic fixed template. The composition, typography, image styling, and layout should be designed around the brand mood: [BRAND MOOD]. Main colors should follow [COLOR PALETTE].

The visual style should be clean, elevated, editorial, premium, fashion-forward, and highly designed. Use realistic fashion photography aesthetics, refined lighting, strong negative space, elegant composition, and a clear hierarchy between image, typography, and white space. Keep the overall feeling magazine-like, brand-driven, and visually collectible.

Use [DIRECTION TYPE] as the layout structure:

If Giant typography campaign: create a bold editorial poster with oversized typography as a major visual element. Let the headline [HEADLINE] interact with the model or product. Use powerful type placement, strong composition, and a premium campaign feel.

If Couture magazine cover: create a refined fashion cover with a strong central figure or hero product, elegant masthead-style typography, and a polished couture editorial mood. Add a few short editorial text elements such as [SUBHEADLINE] and [EXTRA SMALL TEXT].

If Accessories editorial page: create a fashion accessories-focused editorial page featuring [PRODUCT OR SUBJECT], with a layout that feels like a premium magazine spread or branded product story. The product should be clearly featured, with elegant typography and supporting editorial text.

If Lookbook collage spread: create a stylish editorial collage layout using multiple fashion images within one page, arranged like a premium lookbook spread. Keep the collage controlled, minimal, and fashion-oriented, not messy. Use [HEADLINE] and [SUBHEADLINE] as the key editorial text.

Typography should feel intentional and premium. Use large elegant serif or fashion-style typography, with smaller supporting editorial text. Main headline: [HEADLINE]. Subheadline: [SUBHEADLINE]. Optional small text: [EXTRA SMALL TEXT]. Text should be clean and readable, and should support the visual instead of overpowering it.

If a model is included, use [MODEL DESCRIPTION] and style the fashion image to match the brand mood. The model should look editorial, believable, and premium, not like a generic commercial stock model. If the image is product-focused, make the product the hero and support it with strong layout and campaign styling.

The final image should look suitable for brand moodboards, campaign mockups, early brand concepts, editorial poster design, social media visuals, or lookbook exploration.

Avoid: cheap e-commerce look, template-looking layouts, cluttered composition, low-end poster design, excessive text, random decorations, childish styling, weak typography, stock-photo feeling, messy collage, cartoon look, unrealistic anatomy, deformed hands, distorted products, bad text rendering, and real copyrighted magazine logos.

Generate one finished image only in [ASPECT RATIO].
```

## 四种版式方向

### 1. Giant typography campaign
大字报版式，超大标题作为主要视觉元素，与模特/产品互动

### 2. Couture magazine cover
杂志封面版式，中心人物/产品 + 优雅报头风格字体

### 3. Accessories editorial page
配饰版式，产品为主角 + 精致排版支撑

### 4. Lookbook collage spread
Lookbook拼贴版式，多图组合但保持克制简约

---

## 风格锚点（必须保持）

- 干净、精致、高端
- 杂志级摄影美学
- 强负空间、优雅构图
- 清晰的图文层级
- 品牌驱动感

---

## 🚫 避免风格

- 廉价电商感
- 模板化布局
- 杂乱构图
- 弱字体设计
- 库存照片感
- 卡通风格
- 变形/扭曲的产品
