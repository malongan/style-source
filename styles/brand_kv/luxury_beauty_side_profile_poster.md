## 一句话理解

高端美妆 campaign 的核心公式：近景侧面女性肖像 + 超大半透明衬线字体 + 红色手写体点缀 + 温暖逆光 = 干净、光泽、电影感的商业级大片。

---

## 核心特点

- **Close-up side-profile female portrait** — 近景侧面女性肖像作为画面主体
- **Realistic skin texture** — 真实皮肤纹理，自然裸妆
- **Warm reverse light** — 温暖逆光，营造光泽感
- **Oversized translucent serif typography** — 超大半透明衬线字体
- **One red handwritten script accent** — 一处红色手写体强调
- **Minimal blurred background** — 极简模糊背景
- **Small brand logo + slogan + copy** — 品牌 logo + 标语 + 正文 + 页脚细节

## 完整模板

```
Prompt:

Create a single vertical 9:16 luxury beauty campaign poster.

Brand: [BRAND NAME]
Subtitle: [BRAND SUBTITLE]
Main title: [PROFILE / GLOW / VEIL / ECLAT]
Red script word: [refine / luminous / softly / timeless]
Slogan: [SHORT SLOGAN]
Main copy: [MAIN COPY]
Portrait angle: [left side profile / right side profile / semi-profile / lowered side profile]
Color palette: [cream beige / pearl white / champagne gold / misty rose / soft silver]
Key focus: [jawline / skin glow / neck / fragrance / pearl earring / collarbone]

Use a close-up side-profile female portrait with realistic skin texture, natural makeup, soft lips, elegant neck and collarbone, natural hair strands, and warm reverse light. Create a premium editorial beauty-ad look with a minimal blurred background, oversized translucent serif typography, one red handwritten script accent, small brand logo, slogan, main copy and footer details.

The image should feel like a high-end beauty, skincare, fragrance or jewelry campaign poster: clean, luminous, refined, cinematic and commercial-grade.

Avoid Chinese text, cheap promo style, plastic skin, distorted hands, repeated faces, messy text, cluttered background and overdone AI beauty retouching.
```

## 风格锚点（必须保持）

- 近景侧面女性肖像（side-profile portrait）
- 真实皮肤纹理，自然裸妆

- 超大半透明衬线字体

## 🚫 避免风格

- 中文文字
- 廉价促销风格
- 塑料感皮肤（plastic skin）
- 变形的手（distorted hands）
- 重复的脸（repeated faces）
- 杂乱的文字（messy text）
- 杂乱的背景（cluttered background）
- 过度的 AI 美颜修图（overdone AI beauty retouching）
