## 一句话理解

HUD 规格面板 × 液态金属大字，技术信任感爆表的产品海报模板。

---

## 核心特点

- **霓虹黄绿渐变背景** — neon yellow-green 到 mint 渐变底，高饱和高对比，科技感十足
- **四边 HUD 规格面板** — 左右两侧布满工程规格表、系统日志、路线状态、烘焙指数等数据面板，赋予产品专业认证感
- **中心 3D 液态金属大字** — 一个大的动词/名词，3D chrome 液态金属质感，居中炸裂/溅射状，重叠在产品上
- **写实产品居中** — 产品照片或人物居中被 chrome 文字重叠覆盖，产品本身写实质感
- **粉色贴纸点缀** — 角落里粉色圆形贴纸 + 2 行状态文字，增加品牌趣味性
- **边缘细节** — 角落十字准星、底部条码、注册标记，强化工业设计感

## 完整模板

```
生成一张横版 16:9 的产品概念海报。

背景为{渐变色}霓虹渐变底色。

画面中心是一个极具质感的光学写实{产品}照片（或{人物}），放置于画面中央偏下位置。

在画面正中央，一个巨大的 3D 液态金属 chrome 质感{核心文字}单词以炸裂/溅射状大字覆盖在产品之上。文字材质为高反光铬金属，带有蓝色和紫色的环境反射光，立体感极强。

画面左右两侧布满HUD风格数据面板和规格表，包含{数据类型}等工程规格数据。表格样式为科技感深色半透明底 + 霓虹绿/青色文字，带微发光。

角落里有一个粉色圆形贴纸，上面有2行状态文字：「{贴纸文字第一行} / {贴纸文字第二行}」

角落有十字准星元素，底部有条码和注册标记（®）。

视觉风格：HUD科技产品海报、霓虹黄绿渐变、液态金属3D大字、工程规格面板、工业设计感、高级质感。

不要极简布局，不要白底，不要柔光，不要无面板数据，不要衬线字体，不要水印，不要真实Logo，不要电商广告感，不要卡通风，不要乱码。
```

## 变量使用指南

| KV套用时的变量 | 对应风格文件变量 | 说明 |
|---------------|-----------------|------|
| 画面-渐变 | {渐变色} | 背景渐变，如：neon yellow-green 到 mint / 霓虹紫到深蓝 / 橙红到暗红 |
| 画面-产品 | {产品} | 中心产品，如：意式咖啡机 / 机械键盘 / 跑鞋 / 相机 / 耳机 |
| 画面-人物 | {人物} | 可选，替代产品的人物，如：赛车手 / 程序员 / 咖啡师 |
| 文字-核心 | {核心文字} | 液态金属大字，产品核心承诺，如：BREW / TYPE / DRIVE / SHOT / CODE |
| 数据-类型 | {数据类型} | 规格面板数据主题，如：咖啡萃取数据 / 键盘触发参数 / 发动机指标 |
| 贴纸-第一行 | {贴纸文字第一行} | 粉色贴纸第一行，如：REVIEWED / APPROVED / TESTED |
| 贴纸-第二行 | {贴纸文字第二行} | 粉色贴纸第二行，如：BY EXPERTS / EDITION / V2.0 |
