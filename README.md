# Style Source — 风格提示词源文件仓库

> 风格提示词的「唯一真相之源」。所有 .md 风格文件在此维护，CI 自动生成画廊数据。

## 快速开始

```bash
git clone git@github.com:malongan/style-source.git
cd style-source
```

## 目录结构

```
├── styles/              ← 所有风格 .md 源文件
├── data/                ← 自动生成的 styles.json（CI 维护）
├── scripts/             ← 工具脚本
└── .github/workflows/   ← CI/CD 配置
```
